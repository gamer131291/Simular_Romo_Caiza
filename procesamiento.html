Paso 3: Procesamiento en el Servidor Web

El servidor web (p. ej. Apache en XAMPP) recibe la solicitud HTTP y determina que recurso devolver.

Como funciona el servidor?

  1. Recibe la peticion HTTP del navegador.
  2. Interpreta la ruta solicitada (p. ej. /index.html).
  3. Busca el recurso en el sistema de archivos o en la base de datos.
  4. Ejecuta scripts del lado del servidor si es necesario.
  5. Construye la respuesta HTTP con codigo de estado y cuerpo.

Ejemplo de respuesta HTTP del servidor

HTTP/1.1 200 OK
Date: Mon, 26 May 2026 18:00:00 GMT
Server: Apache/2.4.57
Content-Type: text/html; charset=UTF-8
Content-Length: 2456
