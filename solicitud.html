Paso 2: La Solicitud HTTP

Una vez que el navegador conoce la IP del servidor, envia una solicitud HTTP (HyperText Transfer Protocol). Los dos metodos mas comunes son GET y POST.

Metodos HTTP principales

Tabla 2. Metodos HTTP GET y POST
MetodoDescripcionCuando se usaEjemplo
GETSolicita datos sin modificarlosCargar paginas, busquedasGET /index.html HTTP/1.1
POSTEnvia datos al servidor para procesarFormularios, loginPOST /login HTTP/1.1
PUTActualiza un recurso existenteEdicion de datos (APIs REST)PUT /usuario/5 HTTP/1.1
DELETEElimina un recurso del servidorBorrado de registrosDELETE /articulo/3 HTTP/1.1

Ejemplo de cabecera HTTP GET

GET /index.html HTTP/1.1
Host: www.utpl.edu.ec
Accept: text/html
Accept-Language: es-EC
User-Agent: Mozilla/5.0
